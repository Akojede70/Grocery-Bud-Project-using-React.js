# Grocery-Bud-Project-using-React.js

The Grocery Bud shows the list of Grocery you can add to the list, and once you add Item and click on the submit button you will be notified on the screen that Item has been Added, you can also Edit the Item that you add to the list, you will also be notify that the value has change once you submit after you edit. You can also delete the Item you add to the list and also clear all the Item in the list.
